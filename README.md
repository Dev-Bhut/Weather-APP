# Weather-APP
A simple weather app built using HTML, CSS, and JavaScript that fetches real-time weather data from the OpenWeatherMap API. 
🌦️ Weather App Project Description
The Weather App is a simple and responsive web application designed to provide real-time weather updates for any city across the globe. Built using HTML, CSS, and JavaScript, this project integrates the OpenWeatherMap API to fetch and display accurate weather data based on user input.

🔧 Features
🔍 City-based Weather Search
Users can enter any city name to get current weather conditions.

🌡️ Real-Time Data
The app fetches up-to-date weather information using the OpenWeather API.

📊 Detailed Weather Info
Displays temperature, weather conditions (e.g., sunny, cloudy, rain), humidity, and wind speed.

🎨 Clean and Responsive UI
Built with HTML and CSS to ensure a smooth user experience on both desktop and mobile devices.

⚙️ Technologies Used
HTML – for structuring the application layout

CSS – for styling the user interface

JavaScript – for handling API requests and dynamic content updates

OpenWeatherMap API – to get weather data based on city input

🚀 How It Works
User types a city name in the search input.

JavaScript captures the input and makes a fetch request to the OpenWeatherMap API.

The API responds with current weather data.

The app displays the weather details dynamically on the page.
